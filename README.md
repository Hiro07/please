# please(1)

## NAME
  please - executes nothing, but you can mention "please" to any commands.

## SYNOPSIS
  please [command ... ]
  
## DESCRIPTION
  This command executes nothing, but you can mention "please" to any commands.

## EXAMPLES
    $ please ls
  
  Return the result of ls command.
